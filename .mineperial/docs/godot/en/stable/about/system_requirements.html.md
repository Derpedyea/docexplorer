**System requirements**

This page lists the minimum and recommended hardware and software requirements for running the Godot Editor itself and for running a project that has been exported from it.

---

## Godot editor

### Desktop or laptop PC – Minimum

| Component | Minimum |
|---|---|
| Operating system | Windows 7 or newer, macOS 10.13 or newer, Linux (64‑bit) |
| CPU | 2.0 GHz 64‑bit processor |
| RAM | 4 GB |
| GPU | OpenGL 3.3+ or Vulkan support |
| Storage | 200 MB free disk space |

### Desktop or laptop PC – Recommended

| Component | Recommended |
|---|---|
| Operating system | Windows 10 or newer, macOS 12 or newer, recent Linux |
| CPU | 3.0 GHz 64‑bit processor or better |
| RAM | 8 GB |
| GPU | Dedicated GPU with at least 1 GB VRAM and OpenGL 3.3/Vulkan support |
| Storage | 200 MB free disk space |

### Mobile device (smartphone / tablet) – Minimum

| Component | Minimum |
|---|---|
| Android | 5.0 (Lollipop) or newer |
| iOS | 12 or newer |
| GPU | OpenGL ES 3.0 or higher |

### Mobile device (smartphone / tablet) – Recommended

| Component | Recommended |
|---|---|
| Android | 9+ |
| iOS | 15+ |
| GPU | OpenGL ES 3.2 or higher (Vulkan if available) |

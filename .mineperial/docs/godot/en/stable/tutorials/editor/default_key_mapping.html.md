# Default Editor Shortcuts

Godot’s editor exposes many keyboard shortcuts that help speed up workflow. The following sections list the shortcuts that are active by default.  (You can customize any of them from **Editor → Settings → Input Map**.)

> **Tip:**  On macOS, replace the *Ctrl* key with *⌘* and the *Alt* key with *⌥*.

---

## General Editor Actions

| Action | Windows / Linux | macOS |
|--------|-----------------|-------|
| New Project | **Ctrl + N** | **⌘ + N** |
| Open Project | **Ctrl + O** | **⌘ + O** |
| Save Project | **Ctrl + S** | **⌘ + S** |
| Save All | **Ctrl + Shift + S** | **⌘ + Shift + S** |
| Undo | **Ctrl + Z** | **⌘ + Z** |
| Redo | **Ctrl + Y** | **⌘ + Y** |
| Duplicate | **Ctrl + D** | **⌘ + D** |
| Delete | **Del** | **Del** |
| Reopen Closed Tab | **Ctrl + Shift + T** | **⌘ + Shift + T** |
| Toggle Fullscreen | **F11** | **⌘ + Ctrl + F** |
| Toggle Debugger | **Alt + 6** | **⌥ + 6** |
| Toggle Help | **F1** | **F1** |

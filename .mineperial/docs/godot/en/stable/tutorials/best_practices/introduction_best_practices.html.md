# Introduction

This series is a collection of best practices to help you work efficiently with Godot. Godot allows for a great amount of flexibility in how you structure a project’s codebase and break it down into logical components, but following a consistent set of conventions and architectural patterns can make your projects more maintainable, easier to debug, and more scalable.

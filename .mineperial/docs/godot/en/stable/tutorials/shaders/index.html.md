# Shaders

The **Shaders** section of the Godot documentation contains everything you need to learn, create, and manage shaders in Godot.  
It is organized into a series of tutorials, references, and style guidelines that walk you through:

- **Introduction to shaders** – concepts, types, and the Godot shading language.  
- **Shading reference** – a complete API reference for all built‑in shader functions and built‑ins.  
- **Your first shader** – step‑by‑step example of writing a simple fragment shader.  
- **Using VisualShaders** – creating shaders graphically with the Visual Shader editor.  
- **Using compute shaders** – writing GPU‑side compute code in Godot.  
- **Screen‑reading shaders** – shaders that sample the rendered screen.  
- **Converting GLSL to Godot shaders** – guidance for porting legacy GLSL code.  
- **Shaders style guide** – best practices for naming, organization, and performance.  
- **Using a Sub‑shader** – composing shaders in layers.

**Using NavigationPaths**

This page provides a guide to obtaining and using `NavigationPath` objects in Godot. It covers:

- Getting a navigation path from the `NavigationServer`.
- Querying paths from a navigation map that has a navigation mesh.
- Basic code examples and tips for working with `NavigationPath` objects.

*(The full documentation includes detailed sections, code snippets and further reading. For the complete text, see the official Godot Engine documentation: https://docs.godotengine.org/en/stable/tutorials/navigation/navigation_using_navigationpaths.html)*

## Overview

Godot allows you to extend the editor and project runtime with plugins written in GDScript, C#, or GDExtension.  
These tutorials cover:

- **Editor plugins** – Creating, installing, and using plugins that run inside the Godot editor.  
- **Running code in the editor** – Executing scripts that interact with the editor, modify the scene tree, and provide custom tools.

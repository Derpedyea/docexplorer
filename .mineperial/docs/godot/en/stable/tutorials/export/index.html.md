## How to use this section

1. **Start with the “Exporting projects” page** for a general overview of the export process, including setting up export templates and configuring export presets.
2. Move to **platform‑specific pages** to learn how to export to a particular target (e.g., Windows, Linux, Android, iOS).
3. Use the **“Feature tags”** page to understand which export features are available on which platforms.
4. For advanced customization or troubleshooting, consult the pages on **packs, patches, and mods** or the various **platform‑specific guides**.

Feel free to jump straight to the platform that interests you – all export presets and settings are described in a consistent, cross‑platform manner.

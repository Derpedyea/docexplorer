**Asset Library – Godot Engine (stable) documentation**

The Godot Asset Library is a public repository for user‑contributed content such as scenes, scripts, shaders, and other resources that can be imported directly into a Godot project.

---

## 1. About the Asset Library

- The Asset Library is maintained by the community and allows you to share and download reusable assets.
- Assets are grouped by type and tagged with keywords to help you find what you need.
- Each asset page contains a description, a list of tags, a rating, and download statistics.

**Links**

- [Asset Library homepage](https://godotengine.org/asset-library/)

---

## 2. Using the Asset Library

1. **Open the Asset Library in the editor**  
   • In the Godot editor, click *Project → Asset Library* from the menu or open the Asset Library panel.  
2. **Search and filter assets**  
   • Use the search box or the filter panel to narrow results by tags, category, or popularity.  
3. **Download and import**  
   • Click an asset to view its details, then press *Download* or *Install* to add it to your project.  
4. **Install dependencies**  
   • Some assets require other packages or scripts. The asset page lists any required dependencies and will prompt you to install them automatically.

---

## 3. Submitting to the Asset Library

1. **Create a project or scene** that you want to share.
2. **Prepare the asset**  
   • Ensure it is well organized, contains a `metadata.json` (if applicable), and follows the [Asset Library submission guidelines](https://docs.godotengine.org/en/stable/community/asset_library/submitting.html).  
3. **Upload your asset**  
   • Log in to your Godot account and click *Submit* from the Asset Library panel.  
4. **Fill out the submission form**  
   • Provide a title, description, tags, and a preview image.  
5. **Publish** – Once approved, your asset will appear in the library and become available for download by other users.

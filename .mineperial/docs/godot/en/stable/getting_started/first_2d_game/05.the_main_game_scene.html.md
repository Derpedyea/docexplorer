# The main game scene

After creating the player and enemy scenes, it’s time to combine everything into a playable main scene.

## 1. Create the main scene

1. In the **Scene** dock, click **+** and select **Node**.  
2. Rename the root node to `Main`.  
3. Save the scene (`Ctrl+S`) as `Main.tscn`.

### 1.1 Add child nodes

Add the following children to `Main`:

| Node | Type | Description |
|------|------|-------------|
| `Player` | **Instance of Player scene** | Drag the `Player.tscn` file onto the `Main` node. |
| `SpawnTimer` | **Timer** | A timer that controls enemy spawn intervals. |
| `EnemyContainer` | **Node** | Holds spawned enemies to keep the scene tree tidy. |
| `HUD` | **Instance of Heads‑Up Display** | (Optional) If you’re adding a HUD later. |

## 2. Spawning mobs

The `SpawnTimer` will call a function that creates new enemies.  
Add this script to the `Main` node.

```gdscript
# main.gd
extends Node

# Load the enemy scene
var EnemyScene = preload("res://enemy.tscn")

# Keep track of spawned enemies
var enemies = []

@onready var spawn_timer = $SpawnTimer

func _ready() -> void:
    # Configure the spawn timer
    spawn_timer.wait_time = 2.0
    spawn_timer.connect("timeout", self, "_on_SpawnTimer_timeout")
    spawn_timer.start()

func _on_SpawnTimer_timeout() -> void:
    # Instantiate a new enemy
    var enemy = EnemyScene.instantiate()
    enemy.position = get_random_spawn_position()
    enemies.append(enemy)
    $EnemyContainer.add_child(enemy)

func get_random_spawn_position() -> Vector2:
    # Randomly choose a spawn point from a predefined array
    var spawn_points = [
        Vector2(0, 0),
        Vector2(200, 0),
        Vector2(400, 0),
        Vector2(600, 0)
    ]
    return spawn_points[randi() % spawn_points.size()]
```

> **Tip:** Keep the enemy scenes inside an `EnemyContainer` node to avoid cluttering the main scene tree.

## 3. Main script details

Below is a more complete version of the script that also cleans up enemies that go off‑screen:

```gdscript
extends Node

var EnemyScene = preload("res://enemy.tscn")
var enemies = []

@onready var spawn_timer = $SpawnTimer
@onready var enemy_container = $EnemyContainer
@onready var player = $Player

func _ready() -> void:
    spawn_timer.wait_time = 2.0
    spawn_timer.connect("timeout", self, "_on_SpawnTimer_timeout")
    spawn_timer.start()

func _on_SpawnTimer_timeout() -> void:
    var enemy = EnemyScene.instantiate()
    enemy.position = get_random_spawn_position()
    enemies.append(enemy)
    enemy_container.add_child(enemy)

func get_random_spawn_position() -> Vector2:
    var spawn_points = [
        Vector2(0, 0),
        Vector2(200, 0),
        Vector2(400, 0),
        Vector2(600, 0)
    ]
    return spawn_points[randi() % spawn_points.size()]

func _process(delta):
    for enemy in enemies:
        if enemy.is_off_screen():
            enemy.queue_free()
            enemies.erase(enemy)
```

> **Note:** `is_off_screen()` should be implemented in the enemy script to return `true` when it moves beyond the visible area.

## 4. Testing the scene

1. Save all scenes.  
2. In Project Settings → **Application** → **Run → Main Scene**, set `res://Main.tscn` as the main scene.  
3. Click the **Play** button.  
4. You should see the player and enemies spawning at regular intervals.

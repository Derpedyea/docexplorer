### Description
`EditorFileSystem` is a node that holds information about all resources in the editor’s filesystem, including their types, paths, and associated metadata. It is used internally by the editor to provide quick access to project assets and to expose file system changes to the editor UI.

> **Note:** This class is not intended for direct use in game scripts; it is an internal part of the editor.

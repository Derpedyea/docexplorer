**NoiseTexture2D**

*Godot 4.x class reference – 2‑D texture filled with procedural noise.*

---

### Overview
`NoiseTexture2D` is a 2‑D texture that uses a `Noise` resource to generate a procedural texture at runtime. It can be used as a material source, a sprite texture, or any other place where a `Texture2D` is expected. The underlying noise is generated by the FastNoiseLite library (or an equivalent noise implementation) and is updated automatically when the noise parameters change.

```
class NoiseTexture2D : Texture2D
```

---

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `noise` | `Noise` | – | The `Noise` resource that produces the pixel values. |
| `width` | `int` | 256 | Width of the generated texture in pixels. |
| `height` | `int` | 256 | Height of the generated texture in pixels. |
| `offset` | `Vector2` | `Vector2.ZERO` | Horizontal and vertical offset applied before sampling the noise. |
| `scale` | `float` | 1.0 | Scale applied to the noise coordinates (affects frequency). |
| `seed` | `int` | 0 | Random seed used by the noise generator. |
| `octaves` | `int` | 1 | Number of octaves for fractal noise. |
| `persistence` | `float` | 0.5 | Persistence of the fractal noise. |
| `lacunarity` | `float` | 2.0 | Lacunarity of the fractal noise. |
| `use_mipmaps` | `bool` | true | Whether to generate mipmaps for the texture. |

*(Only a subset of the most common properties is shown; the full list can be found in the official class reference.)*

### Methods

| Method | Return Type | Parameters | Description |
|--------|-------------|------------|-------------|
| `set_noise(noise: Noise)` | `void` | `noise` | Assigns a new `Noise` resource and updates the texture. |
| `get_noise()` | `Noise` | – | Returns the currently assigned noise resource. |
| `update()` | `void` | – | Forces a regeneration of the texture based on the current parameters. |

---

### Signals

| Signal | Parameters | Description |
|--------|------------|-------------|
| `changed()` | – | Emitted when the texture is regenerated (e.g., after changing a property). |

---

### Example: Creating a Procedural Heightmap

```gdscript
var noise = FastNoiseLite.new()
noise.seed = 123
noise.frequency = 0.02

var texture = NoiseTexture2D.new()
texture.noise = noise
texture.width = 512
texture.height = 512
texture.update()

var sprite = Sprite2D.new()
sprite.texture = texture
add_child(sprite)
```

The example above creates a 512×512 noise texture that can be displayed by a `Sprite2D`.

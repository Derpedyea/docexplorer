**PlaceholderCubemapArray**

*Inherits:* `PlaceholderTextureLayered< TextureLayered< Texture<Resource<RefCounted<Object>>> >`

A **CubemapArray** without image data. This class replaces a `CubemapArray` or a `Cubemap` when no actual image data is available, providing a lightweight placeholder that can be used in the editor or during runtime until a proper texture is loaded or generated.

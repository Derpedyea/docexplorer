## Class Overview

`AudioStream` is an abstract base class that represents an audio stream in Godot.  
It provides a common interface for all audio formats and streaming mechanisms used in the engine. Concrete subclasses implement the specifics for different audio sources (e.g., MP3, OGG, microphone, generated audio, playlists, etc.).

## Example Usage (GDScript)

```gdscript
var audio_stream : AudioStream = preload("res://music.ogg")

var player = AudioStreamPlayer.new()
player.stream = audio_stream
player.volume_db = -6.0
add_child(player)

player.play()
```

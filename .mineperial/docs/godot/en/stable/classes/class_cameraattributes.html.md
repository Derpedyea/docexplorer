# CameraAttributes

> *Parent class for camera settings.*

## Inheritance

- **Base class:** `Resource` (`RefCounted<Object>`)
- **Inherited By:**
  - `CameraAttributesPhysical`
  - `CameraAttributesPractical`

## Description

`CameraAttributes` is a resource that controls camera-specific attributes, such as depth of field, motion blur, lens distortion, and other visual effects that can be applied to a `Camera3D` or `Camera2D`.  
It acts as a common interface for both practical and physical camera implementations, allowing the editor and scripts to modify camera behaviour in a unified way.

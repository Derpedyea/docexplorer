**SpringBoneCollision3D**  
==========================

> *Godot Engine (stable) – Class reference*

---

### Inheritance
- **Base class:** `Node3D`
- **Inherited by:**
  - `SpringBoneCollisionCapsule3D`
  - `SpringBoneCollisionPlane3D`
  - `SpringBoneCollisionSphere3D`

### Description
`SpringBoneCollision3D` is a **base class** that provides collision handling for spring‑bone simulations.  
It offers the minimal API required by the various concrete collision shapes (capsule, plane, sphere) that are used by the spring‑bone system.

> - [SpringBone] – The node that drives the spring‑bone physics.

> - [SpringBoneCollisionPlane3D] – Plane collision shape.  
> - [SpringBoneCollisionSphere3D] – Sphere collision shape.

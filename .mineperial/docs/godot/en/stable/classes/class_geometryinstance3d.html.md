# GeometryInstance3D

**Inherits:** `VisualInstance3D` → `Node3D` → `Node` → `Object`

**Description**  
Base node for geometry. The class is intended to provide common functionality for 3D geometry–related nodes. It is a part of the Godot 3D rendering pipeline and is used by classes such as `CPUParticles3D`, `CSGShape3D`, `GPUParticles3D`, `Label3D`, `MeshInstance3D`, `MultiMeshInstance3D`, and `SpriteBase3D`.

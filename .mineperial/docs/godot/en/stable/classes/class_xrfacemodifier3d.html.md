# XRFaceModifier3D

**Experimental** – this class may be changed or removed in future versions.  
**Inherits**: `Node3D` (`Node<Object>`).

A node that drives standard face meshes using the weights provided by an `XRFaceTracker`.  
The class is part of Godot’s experimental XR toolkit and is intended for use with face‑tracking hardware to animate 3‑D character meshes in real time.  

> *For detailed API reference, including all properties, methods and signals, visit the official Godot documentation for the current stable version.*

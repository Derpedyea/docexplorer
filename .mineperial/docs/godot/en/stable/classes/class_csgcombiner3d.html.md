# CSGCombiner3D

**Inherits:** `CSGShape3D<GeometryInstance3D<VisualInstance3D<Node3D<Node<Object`

## Description
A **CSGCombiner3D** node allows you to combine other CSG modifiers into a single, unified shape. It is especially useful for creating complex arrangements of constructive solid geometry (CSG) objects in 3D scenes.

**CapsuleMesh**  
*Godot Engine 4.0/4.1 class reference*  

---

### Overview
`CapsuleMesh` is a built‑in primitive mesh class that represents a 3‑D capsule (a cylinder with hemispherical caps).  
It inherits from `PrimitiveMesh`, which in turn inherits from `Mesh` → `Resource` → `RefCounted` → `Object`.

> **Description**  
> Class representing a capsule‑shaped `PrimitiveMesh`.  

---

### Properties
| Name | Type | Default | Description |
|------|------|---------|-------------|
| `radius` | `float` | `0.5` | Radius of the capsule's cylindrical section. |
| `height` | `float` | `1.0` | Height of the cylindrical section (excluding the hemispheres). |
| `radial_segments` | `int` | `16` | Number of subdivisions around the capsule’s axis. |
| `rings` | `int` | `8` | Number of subdivisions along the capsule’s height. |

All properties are *editable* in the Godot editor and can be accessed from script.

---

### Methods
| Method | Return type | Parameters | Description |
|--------|-------------|------------|-------------|
| `get_radius()` | `float` | – | Returns the current radius. |
| `set_radius(value: float)` | `void` | `value: float` | Sets the radius. |
| `get_height()` | `float` | – | Returns the current height. |
| `set_height(value: float)` | `void` | `value: float` | Sets the height. |
| `get_radial_segments()` | `int` | – | Returns the number of radial segments. |
| `set_radial_segments(value: int)` | `void` | `value: int` | Sets the radial segments. |
| `get_rings()` | `int` | – | Returns the number of rings. |
| `set_rings(value: int)` | `void` | `value: int` | Sets the number of rings. |

These methods are automatically generated by Godot’s property system, but they can also be called explicitly from scripts if needed.

---

### Signals
`CapsuleMesh` does not define any custom signals; it only inherits those from its parent classes.

---

### Example (GDScript)

```gdscript
var capsule := CapsuleMesh.new()
capsule.radius = 0.75
capsule.height = 1.2
capsule.radial_segments = 24
capsule.rings = 12

var mesh_instance := MeshInstance3D.new()
mesh_instance.mesh = capsule
add_child(mesh_instance)
```

This script creates a capsule mesh at runtime, adjusts its dimensions, and attaches it to a `MeshInstance3D` node so it can be rendered in the scene.

---

### Example (C#)

```csharp
var capsule = new CapsuleMesh();
capsule.Radius = 0.75f;
capsule.Height = 1.2f;
capsule.RadialSegments = 24;
capsule.Rings = 12;

var meshInstance = new MeshInstance3D { Mesh = capsule };
AddChild(meshInstance);
```

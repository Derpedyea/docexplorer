**CSGPrimitive3D** – Godot Engine (stable) documentation  
===========================================================

> *Base class for all 3‑D constructive solid geometry (CSG) primitive nodes.*  
>  Inherits from:  
>  `CSGShape3D` → `GeometryInstance3D` → `VisualInstance3D` → `Node3D` → `Node` → `Object`

---

> *The `CSGPrimitive3D` class provides a generic interface for all primitive shapes used in Godot’s CSG system. It exposes the common properties and methods shared by all CSG primitives (e.g., `operation`, `material`, `collision_layer`, etc.). The concrete subclasses implement specific geometry.*

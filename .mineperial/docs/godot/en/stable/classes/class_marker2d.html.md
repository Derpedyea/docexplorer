**Marker2D**

**Inheritance**  
`Node2D` → `CanvasItem` → `Node` → `Object`

---

### Description
A generic 2D position hint for editing.  
It behaves exactly like a plain `Node2D`, but is displayed in the editor with a special icon that helps developers place objects in the scene.

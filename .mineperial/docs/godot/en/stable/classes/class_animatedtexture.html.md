# AnimatedTexture

**Deprecated**  
This class does not work properly in current versions and may be removed in the future.  
There is currently no equivalent workaround.

## Inheritance

## Description
`AnimatedTexture` was intended to provide a simple way to play a sequence of textures like a sprite sheet.  
It is no longer supported in modern Godot releases and is kept only for legacy compatibility.

# GDScript

> A script implemented in the GDScript programming language, saved with the `.gd` extension.

### Inherits
```
Script < Resource < RefCounted < Object
```

### Description
`GDScript` is the dedicated scripting language for Godot Engine. It is a dynamically typed language with syntax inspired by Python, designed for rapid development of game logic and user interface behavior within the Godot environment. The language integrates tightly with Godot's scene system, signals, and resource handling.

# Separator

**Abstract base class for separators**, used for separating UI elements in Godot.

---

## Inheritance

```
Separator
 └─ Control
      └─ CanvasItem
           └─ Node
                └─ Object
```

---

## Inherited by

- **HSeparator** – Horizontal separator
- **VSeparator** – Vertical separator

---

## Description

`Separator` provides the foundational functionality for UI separators that divide or delineate sections of a layout. Concrete subclasses such as `HSeparator` and `VSeparator` implement the specific orientation and visual style.

> *Note: This page only covers the abstract base class. For more details on the concrete separator types, see the respective pages for **HSeparator** and **VSeparator**.*

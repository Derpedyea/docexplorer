# OmniLight3D

**Omnidirectional light, such as a light bulb or a candle.**  
An `OmniLight3D` is a type of `Light3D` that emits light equally in all directions.

## Inheritance

```
OmniLight3D
└─ Light3D
   └─ VisualInstance3D
      └─ Node3D
         └─ Node
            └─ Object
```

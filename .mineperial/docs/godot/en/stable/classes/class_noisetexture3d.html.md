# `NoiseTexture3D`

> *Inherits:* `Texture3D` → `Texture` → `Resource` → `RefCounted` → `Object`

`NoiseTexture3D` is a 3D texture that is automatically filled with procedural noise generated by a `Noise` resource (FastNoiseLite or an alternative noise generator). It can be used wherever a 3D texture is required – for volumetric effects, procedural materials, or data‐driven shaders.

---

## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `size` | `Vector3i` | `(0, 0, 0)` | The dimensions of the 3‑D texture. Each component must be ≥ 1. |
| `noise` | `Noise` | – | The noise resource that generates the values. |
| `generate_mipmaps` | `bool` | `false` | If `true`, generates mipmap levels when the texture is uploaded. |
| `repeat` | `bool` | `false` | Whether the texture repeats along the axes. |
| `period` | `float` | `1.0` | The spatial period of the noise pattern (controls frequency). |
| `offset` | `Vector3` | `(0, 0, 0)` | Global offset applied before sampling the noise. |
| `lacunarity` | `float` | `2.0` | Controls octave frequency growth. |
| `gain` | `float` | `0.5` | Controls octave amplitude decay. |
| `octaves` | `int` | `1` | Number of noise octaves. |
| `seed` | `int` | `0` | Seed used for the noise generator. |
| `frequency` | `float` | `1.0` | Base frequency of the noise. |
| `persistence` | `float` | `0.5` | Base persistence (amplitude scaling). |

> **Tip:** Most of these properties mirror those of `NoiseTexture2D`, except that `size` is a 3‑D vector.

## Example Usage

```gdscript
# Create a new 3D noise texture of size 64×64×64
var noise_tex = NoiseTexture3D.new()
noise_tex.size = Vector3i(64, 64, 64)

# Configure the underlying noise resource
var noise = FastNoiseLite.new()
noise.noise_type = FastNoiseLite.NOISE_TYPE_PERLIN
noise.seed = 42
noise_tex.noise = noise

# Set additional parameters
noise_tex.period = 8.0
noise_tex.octaves = 4
noise_tex.lacunarity = 2.0
noise_tex.gain = 0.5

# Generate the texture
noise_tex.create()

# Use the texture in a shader
var material = StandardMaterial3D.new()
material.albedo_texture = noise_tex
```

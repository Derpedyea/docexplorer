## Overview

`GLTFDocumentExtensionConvertImporterMesh` is a specialized GLTF document extension used internally by Godot to convert importer‑mesh data into a form suitable for runtime usage. This class is part of the Godot class reference and is intended for advanced users who need to understand or modify the GLTF import pipeline.

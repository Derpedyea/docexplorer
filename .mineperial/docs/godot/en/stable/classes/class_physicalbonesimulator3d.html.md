**PhysicalBoneSimulator3D**

> *Inherits:* `SkeletonModifier3D<Node3D<Node<Object>`

---

### Description
Node that can be the parent of `PhysicalBone3D` and can apply the simulation results to a `Skeleton3D`.  
It serves as a container that drives the physics simulation of a skeleton's bones and updates the skeleton accordingly.

**PortableCompressedTexture2D**  
--------------------------------

> **Inheritance chain**  
> `Texture2D` → `Texture` → `Resource` → `RefCounted` → `Object`

---

### Description
`PortableCompressedTexture2D` provides a compressed texture that can be stored on disk and/or uploaded to VRAM in a portable format. It is useful for assets that need to be kept platform‑agnostic while still taking advantage of GPU‑specific compressed texture formats.
